# Frontend Feature

Implement the frontend for the specified feature.

Requirements:

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Query
- React Hook Form
- Zod

Rules:

- Follow docs/design_system.md and docs/design_tokens.md for all styling.
- No business logic in components.
- No direct fetch().
- Create reusable components.
- Use loading, empty, and error states.
- Follow Tribe branding (dark-first).
- Use semantic token classes (e.g. bg-primary, text-muted) — never raw hex values.
